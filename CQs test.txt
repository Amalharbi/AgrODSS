CQ1: What is the suitable chemical control of (Red palm weevil)?

######### SPARQL query ###############
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX dp: <http://www.owl-ontologies.com/DPDAPO#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX owl: <http://www.w3.org/2002/07/owl#>
PREFIX dc: <http://purl.org/dc/elements/1.1/>
SELECT DISTINCT  ?SName ?Chemical_Control 
WHERE {  ?Pest dp:is_Parasite_Of dp:Date_Palm. 
            {?Pest dp:has_Common_Name  ?SName. FILTER (regex(str(?SName),"Red palm weevil","i")) }
                 ?Pest dp:has_Chemical_Control  ?Chemical_Control}
